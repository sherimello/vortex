## 🎯 Common Questions

### Q: How long does setup take?
**A:** 20-30 minutes if you already have Flutter and an API key. See [SETUP_GUIDE.md](SETUP_GUIDE.md)

### Q: Where do I get an API key?
**A:** https://console.x.ai/api/keys. Instructions in [SETUP_GUIDE.md](SETUP_GUIDE.md#step-2-get-your-grok-api-key)

### Q: How do I use the app?
**A:** Press Ctrl+Q anywhere. Full guide in [README.md](README.md#usage)

### Q: Is it secure?
**A:** Yes! See security details in [README.md](README.md#security-considerations)

### Q: What are system requirements?
**A:** Windows 10+, Flutter SDK. See [README.md](README.md#system-requirements)

### Q: Can I customize it?
**A:** Yes! See customization section in [PROJECT_OVERVIEW.md](PROJECT_OVERVIEW.md)

### Q: What if something breaks?
**A:** Check [README.md](README.md#troubleshooting) troubleshooting section

### Q: Can I use it after closing the app?
**A:** Yes, enable Auto-Start in settings. See [SETUP_GUIDE.md](SETUP_GUIDE.md)
